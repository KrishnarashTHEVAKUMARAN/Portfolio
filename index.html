Mes projets

Jeu de la Vie (Simulation cellule)

JAVA

Le jeu de la vie est un jeu de simulation créé par Horton Conway. En première année de mon cursus en DUT informatique, nous avons eu comme objectif de créer cette application. Elle était sous la forme d’un damier qui contient plein de cellules en vie ou mortes. À chaque génération, les cellules restent en vie ou meurent en fonction des conditions. Comme par exemple: Une cellule en vie ne possédant qu’une cellule voisine en vie meurt par isolement. On devait aussi respecter certaines fonctionnalités données comme par exemple : avoir un bouton avancer de pas à pas ou en avance rapide et un bouton pause pour arrêter l’avance rapide. Cette application est codée en java sous le logiciel Eclipse.

Site E-Commerce pour Misa Crea

PHP_MySQL(XAMPP)_HTML/CSS

Ce projet consistait à programmer une application web d'E-Commerce à l’intention d’un auto-entrepreneur artisan. Le site que nous avons programmé devait permettre à l’administrateur d'exécuter les fonctionnalités suivantes: se connecter et se déconnecter, changer son mot de passe et son profil, gérer les commandes et les demandes des clients, gérer ses stocks et de mettre en ligne ses nouveaux produits. Il devait aussi permettre aux clients de s’inscrire (le login devra être son mail), de récupérer son mot de passe oublié, de changer ses coordonnées sur son profil, de voir son panier et les produits proposés par le site, d’enregistrer un panier provisoire et de procéder au paiement en ligne. La mise en place de cette application web a pour but de faciliter le travail de l’artisan. Ce site est codé en PHP et HTML/CSS/JS sous l'application Visual Studio et Sublime Text. Nous avons utilisé XAMPP pour la gestion de base de données avec PHPMyAdmin.

Site pour la gestion de note

DJANGO_MySQL(XAMPP)_HTML/CSS

Le site de gestion de note est mon projet de premier semestre de mon L3 MIAGE. Ce projet a pour but de se familiariser avec le framework python DJANGO. Nous avons utilisé l'architecture MVC (Model, View and Controller) pour la réalisation de ce site web. Par exemple: models.py comportait les classes nécessaires pour l'exécution de cette application comme les classes élèves et matières. Et views.py permettait d'afficher les pages html. Le site avait pour objectif d'aider l'administrateur de gérer les notes des élèves comme ajouter, modifier, supprimer les notes et afficher le bulletin et la décision de jury des élèves sélectionnés par l'admin (ADMIS, DEFAILLANT et les mentions). Ce site est code en Django et HTML/CSS sous l'IDE PyCharm.
